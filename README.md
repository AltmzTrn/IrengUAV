### IrengUAV
lightweight UAV firmware

# Introduction
Project started in early 2024 by Altamis Tristan to implement direct low-level language in UAV systems. This project aims to create a small, lightweight firmware, to be implemented in lowwer cost microcontrollers such as the STM32F1 & STM32F4 - series development boards, ESP32 Development Boards, and Arduino. The target Airframe for this Firmware will be Small UAVs which falls under the 25kg MTOW limit.

# Current Version
Currently, the only ongoing development of this firmware is for a sub-100g brushed quadcopter using an STM32F103C8T6 as its main flight controller.

# Archived Versions
Development starts as early as January 2024, however, it was just a weekend project at the time and so all of the updates, commits, and versions were updated on Google Drive instead. The archive folder here stores all of the previously developed firmwares for various types of Airframes from 2024 to 2025.
